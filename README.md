# YummieApp-on-UIKit
Creating an application for a restaurant with the ability to add an order.

## Page content
- [Technologies](#technologies)
- [Images](#images)

## Technologies
- Swift 5
- IOS 14.0
- Xcode 13.0
- UIKit
- CocoaPods: Kingfisher, ProgressHUD, IQKeyboardManagerSwift
